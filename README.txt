WEB-приложение "Система управления гостиницей"

Содержит интерфейсы:
1) Пользователя (использование ресурсов гостиницы, заказ номеров)
2) Администратора (управление гостиницей, осуществление контроля)

Функционал приложения:
Пользователь может создать аккаунт, редактировать и удалять персональные данные. 
Основной функционал заключается в предоставлении пользователю возможности 
забронировать путёвку на определённый период времени. Параметры путёвки также могут
быть отредактированы в процессе получения, либо в любое время после начала действия путёвки.
Администратор, соответственно, имеет доступ ко всей информации о гостинице, состоянии 
корпусов и номеров, списке предстоящих путёвок и ихъ клиентов. Администратор может управлять
данными параметрами гостиницы, Добавление-Изменение-Удаление.
